## Usage

- Compile project:
```bash
$ make
```

- **Compare** directories:
```bash
$ ./cmpcat -d folderA folderB
```

- **Merge** directories:
```bash
$ ./cmpcat -d folderA folderB -s outputFolder [-f]
```
**or**
```bash
$ make run
```

- Remove object files & executable program
```bash
$ make clear
```
